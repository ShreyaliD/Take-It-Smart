# 🏠 House Price Prediction (Regression)

## 📌 Overview
Machine Learning project to predict house prices based on features like square footage, bedrooms, bathrooms, lot size, garage size, neighborhood quality, and year built.

---

## 🎯 Objective
- Perform data preprocessing and EDA  
- Train regression models  
- Evaluate and compare model performance  

---

## 📊 Dataset
- Rows: 1000  
- Features: 7 (all numerical)  
- Target: `House_Price`  

---

## ⚙️ Tech Stack
- 🐍 Python  
- 📊 Pandas, NumPy  
- 📈 Matplotlib, Seaborn  
- 🤖 Scikit-learn 
---

## 🔍 Steps Performed
- Data cleaning (missing values, duplicates)  
- Outlier handling (IQR method)  
- Exploratory Data Analysis (EDA)  
- Log transformation  
- Feature scaling  
- Model training  

---

## 🤖 Models
- Linear Regression  
- KNN Regression  

---

## 📈 Results

| Model | MAE | RMSE | R² |
|------|------|------|------|
| Linear Regression | 0.09 | 0.11 | 0.94 |
| KNN Regression | 0.13 | 0.18 | 0.86 |

---

## 🏆 Best Model
**Linear Regression** performed better with:
- Lower error values  
- Higher R² score
